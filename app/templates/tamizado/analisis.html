Análisis de Tamaño de Partícula

Análisis granulométrico con modelo Gates-Gaudin-Schuhmann

Datos de Entrada
T: Peso total de muestra (g) → campo único
Datos de Mallas
Tamaño de partícula di (mm) Peso retenido Ri (g) Acción
Datos de entrada esperados:
  • T: Peso total de muestra (g) → campo único
  • di: Tamaño de partícula (por malla, en mm o μm)
  • Ri: Peso retenido en cada malla (g)
PASOS DE CÁLCULO
1. % Retenido por malla:
%Ri = RiT × 100
2. % Acumulado retenido:
%AcumRi = ∑j=1i %Rj
3. % Acumulado pasante:
%Pi = 100 - %AcumRi
4. Logaritmos (para graficar y obtener n):
log(di), log(%Pi)
5. Regresión lineal en gráfico log-log para obtener n:
log(%Pi) = n × log(di)
