Datos de Mallas
Abertura d (mm) Peso Retenido (g) Acción
Modelo Gates-Gaudin-Schuhmann
Datos de entrada:
  • T: Peso total de muestra (g) → campo único
  • di: Tamaño de partícula (por malla, en mm o μm)
  • Ri: Peso retenido en cada malla (g)
PASOS DE CÁLCULO
1. % Retenido por malla:
%Ri = Ri/T × 100
2. % Acumulado retenido:
%AcumRi = Σ%Rj (j=1 hasta i)
3. % Acumulado pasante:
%Pi = 100 - %AcumRi
4. Logaritmos (para graficar):
log(di), log(%Pi)
5. Regresión lineal log-log:
log(%Pi) = n × log(di)
Usa mínimos cuadrados para obtener la pendiente n
